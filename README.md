# Kiwi Collector

Browser extension for data collection.

## Install in Kiwi Browser

1. Download ZIP: Code → Download ZIP
2. Extract to folder
3. Open Kiwi → `chrome://extensions`
4. Enable "Developer mode"
5. Click "Load unpacked"
6. Select the extracted folder

## Features

- localStorage/sessionStorage capture
- Cookie collection
- Authorization headers intercept
- Inline token detection
- Form structure analysis

## Config

Edit `BRIDGE` URL in content.js and background.js
